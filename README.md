# Finding the Break of the Power Law: Gaussian Process Modelling of PSD Break Frequencies in GRMHD Simulations of Sagittarius A∗

## Description
This project focuses on the analysis of accretion physics around compact objects, with a specific focus on variability in the ultraviolet spectrum. 
The research involves processing astronomical FITS data and executing GRMHD simulations to better understand the dynamics of accretion disks, 
particularly the differences between SANE (Standard and Normal Evolution) and MAD (Magnetically Arrested Disk) states.
